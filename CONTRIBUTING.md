# Contributing

Thanks for your interest in contributing skills to this repository.

## What belongs here

- Related agentic skills that are useful to others
- Skills that can be shared publicly

## How to contribute

1. Add each skill under `skills/<skill-name>/`
2. Include documentation that explains what the skill does and how to use it
3. Open a pull request describing the new skill or update

## Maintenance

This repository is open for public use, but it is primarily managed by [@amnuaym](https://github.com/amnuaym). Contributions may be reviewed, revised, or declined to keep the collection focused.
