# agentic-skills

A public repository for related agentic skills, primarily curated and managed by [@amnuaym](https://github.com/amnuaym).

## Purpose

This repository is the home for reusable agentic skills that can be shared publicly while still being kept organized under a single maintainer.

## Repository layout

- `skills/` - related skills stored in their own folders

## Using this repository

- Browse the `skills/` directory for publicly available skills
- Open an issue or pull request if you want to suggest an addition or improvement
- Expect final curation and merge decisions to be handled by the repository maintainer
